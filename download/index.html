"Nightly" builds

Gisto "nightly" builds are automatic builds triggered by commits to "master" branch.

Please note that these are not stable releases.
Bugs are possible. Those builds lacks the usual installers you may get in the regular releases.

"Nightly" builds for all platforms are available at: build.gistoapp.com

Other install options:

via Homebrew cask: 

SHELL$ brew cask install gisto

via Archlinux AUR: 

SHELL$ yaourt -S gisto

via SlackBuilds.org (Slackware): 

SHELL# sbopkg -i gisto
