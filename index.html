Features

UI themes

Several UI themes available in dark and light flavors.

Themes are not layout based, rather color shades of fonts and elements for now. This may change in the future.

Search

Gists can be filtered by search filter from within the application.

Searching criteria may be gist's description, file name / extention, tag or by public/secret status of a gist.

Tags

Gisto allows to tag each gist with unlimited number of tags.

Tags can be used for simply sorting gist snippets into groups for what makes it easy to search and maintain.

About </Gisto>

The why

Gisto started as an attempt to fulfill the lack of a syntax highlighted and cloud synchronized code snippet solution. You may think of Gisto as Evernote for code.

The who

Gisto is developed by a team of two web developers and uses web standard technologies such as HTML5 and JavaScript.

The how

Gisto made using HTML5/CSS3 it is built with Angular.js on top of Node.js runned by node-webkit. Gisto also uses Angular UI, jQuery, Ace editor, showdown and Font-Awesome
