Rich Editor

Gisto includes Ace a rich code editor for editing your gists on the go and includes features such as syntax highlighting auto-completion emmet and more.

Search

Gists can be found quickly using our search filter and can be filtered by a gist description, file names, tags and more.

Tags

Gisto allows you to tag gists with custom tags this helps you find your gists easily and categorize them upon your needs.

About </Gisto>

why

Gisto started by fulfilling a lack of a syntax highlighted and cloud synchronized code snippet solution.

who

Gisto is developed by a team of two web developers on their spare time.

We appreciate any suggestions or contributions to make Gisto better.

how

Gisto is built using the open web technologies using several open source projects such as AngularJS, node-webkit, Ace Editor and many more
